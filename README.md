# ব্যবসা ম্যানেজমেন্ট অ্যাপ্লিকেশন

এটি একটি Flutter-ভিত্তিক ব্যবসা ম্যানেজমেন্ট অ্যাপ্লিকেশন যা বাংলা ভাষায় ডিজাইন করা হয়েছে।

## বৈশিষ্ট্য

- **ড্যাশবোর্ড**: বিক্রয় সারাংশ এবং চার্ট
- **কাস্টমার ম্যানেজমেন্ট**: গ্রাহক তালিকা পরিচালনা
- **স্টক ম্যানেজমেন্ট**: পণ্য ইনভেন্টরি ট্র্যাকিং
- **বিলিং সিস্টেম**: চালান তৈরি এবং পিডিএফ জেনারেশন
- **ডাটা সিঙ্ক**: স্থানীয় এবং ক্লাউড সিঙ্ক্রোনাইজেশন

## প্রযুক্তি স্ট্যাক

- **Flutter**: UI ফ্রেমওয়ার্ক
- **Hive**: স্থানীয় ডাটাবেস
- **Firebase Firestore**: ক্লাউড ডাটাবেস
- **FL Chart**: ডাটা ভিজুয়ালাইজেশন
- **PDF**: রিপোর্ট জেনারেশন

## ইনস্টলেশন

```bash
flutter pub get
flutter run
```

## সেবা বর্ণনা

### SalesService
বিক্রয় প্রক্রিয়া পরিচালনা করে এবং স্টক আপডেট করে।

### ChartService
মাসিক বিক্রয় ডাটা ভিজুয়ালাইজ করতে বার চার্ট তৈরি করে।

### InvoiceService
পিডিএফ ফরম্যাটে চালান জেনারেট করে এবং প্রিন্ট করে।

### SyncService
ইন্টারনেট সংযোগ পরীক্ষা করে এবং ফায়ারবেসে ডাটা সিঙ্ক করে।
